# Challenges - Native Android App

A native Android application designed to manage and track technical tasks. 

## Technical Stack
* **Language:** Java
* **Framework:** Android SDK
* **Key Components:** Custom `ContentProvider` for local data management, `Intents` for activity navigation, and robust UI structuring.

## Features
* Secure and efficient local data storage using custom providers.
* Clean, native user interface for task management.
* Focus on system-level Android architecture and stable lifecycles.
